Custom Features (User Dashboard "More Features")

User panel এ "More Features" grid এ এই entries live দেখাবে। Hot / New / Upcoming tag লাগাতে পারেন। Type = link হলে external URL খুলবে; type = embed হলে আপনার custom HTML iframe-এ render হবে।

Image URL দিলে built-in icon এর বদলে এই image দেখাবে।

⚠️ HTML / JavaScript সম্পূর্ণ sandboxed iframe-এ চালু হবে। কোনো user data পাবে না।
